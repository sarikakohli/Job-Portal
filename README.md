# MERN Job Portal Application

A full-stack **Job Portal web application** built using the **MERN stack (MongoDB, Express, React, Node.js)**.  
The project follows **industry-standard architecture**, supports **role-based authentication**, and is designed to be **scalable and production-ready**.
